# Parzo

Extract invoice and receipt data from PDFs and export to Excel or CSV in seconds.

## Features

- Upload PDF invoices or receipts
- Automatically extract key data:
  - Supplier
  - Invoice number
  - Date
  - Total & VAT
- Export to CSV or Excel
- No signup required

## Tech

- Next.js
- OpenAI (structured extraction)
- pdf-parse

## Local development

bash pnpm install pnpm dev 

## Notes

Works best with digital PDFs (not scanned images).
